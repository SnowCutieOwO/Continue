# 开箱冷却

在 ExcellentCrates 中，你可以分玩家设置开箱冷却。

你可以通过[编辑器](editor-gui.md)设置开箱冷却。

## 设置

1. 打开编辑器 -> 宝箱（Crates）-> 选择宝箱。
2. **左键**点击**开箱冷却（Open Cooldown）**选项。
3. 在聊天栏中输入单位为秒的冷却。

::: info

若要禁用冷却功能，请将其设置为 `0`。

若要使得宝箱只能开启一次，请将其设置为 `-1`（这样冷却就不会重置）。

:::

::: info

在拥有[绕过权限](permissions.md)的情况下可以直接开启宝箱而无需等待。

:::
